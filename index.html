How to Use

  1. Type or paste your secret text into the "Plaintext" area.
  2. Enter a strong, memorable passphrase.
  3. Set the iterations (250k is a good default).
  4. Click Encrypt Text.
  5. Copy or download the resulting "Output" JSON.
  6. To decrypt, paste the JSON into "Output", enter the *exact* same passphrase, and click Decrypt Text.
