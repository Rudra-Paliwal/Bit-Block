How it works

  1. Select a file from your device.
  2. Enter a strong passphrase and (optionally) increase iterations.
  3. Click Encrypt File to produce a downloadable .secure.json.
  4. To decrypt, choose the produced .secure.json, enter the same passphrase, and click Decrypt File.

Security reminders

  • Everything runs locally in your browser; file contents never leave your machine.
  • Use a strong passphrase and store it securely ‚Äî losing it means losing access to the file.
  • Iterations default to 250,000 ‚Äî raise it for more CPU-hardness if needed.
